# 💻 Portfólio Pessoal

Espaço minimalista e otimizado desenvolvido para apresentar meus projetos e minha evolução como desenvolvedor Full-Stack. O projeto foca em alta performance visual, design responsivo e interatividade fluida.

🔗 **Acesse o portfólio:** https://rabeloodev.github.io/Portf-lio/

## 🛠️ Tecnologias Utilizadas

* **Front-end:** HTML5, CSS3, JavaScript (ES6+), Tailwind CSS
* **Efeitos & Interatividade:** Canvas API (Partículas interativas acionadas via mouse e clique)
* **Ícones:** Lucide Icons

## 🚀 Funcionalidades

* **Bento Grid Layout:** Organização limpa e moderna das informações principais.
* **Fundo Dinâmico:** Sistema de partículas responsivo em 2D que interage diretamente com os movimentos do usuário.
* **Responsividade Total:** Estrutura corrigida e otimizada para garantir visualização perfeita em qualquer dispositivo móvel.
